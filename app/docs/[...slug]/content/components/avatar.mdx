## Usage

Avatar fallbacks are based on the username or full name. For many avatars, use the [StackedAvatars](/docs/components/stacked-avatars) component.

```jsx
import { Avatar } from "natmfat/components/Avatar";

<Avatar
  username="natmfat"
  src="https://www.natmfat.com/logo.png"
  size={tokens.space48}
/>;
```
