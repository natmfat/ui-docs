## Usage

```jsx
import { Button } from "natmfat/components/Button";

<Button>Button</Button>;
```

### Link

You can set the `asChild` parameter and nest the link component to create a link that looks like a button.

```jsx
<Button asChild>
  <Link href="/login">Login</Link>
</Button>
```
